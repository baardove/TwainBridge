Connect TwainBridge.

This address accepts PDF, JPEG, and PNG documents as multipart form data. Use the settings below in TwainBridge, or send a file manually to verify the receiver.

Multipart contract

Accepted fields

File parts are stored. Recognized metadata is shown in the library; additional custom form fields are accepted but intentionally not retained by this demo.

FieldUseRequirement
filePDF, JPEG, or PNG file. Repeat the field for multiple documents.Required
batch_idStable logical ID shared by documents sent together.Recommended
request_idUnique ID for this HTTP attempt and diagnostics.Recommended
document_idStable ID when the request contains one document.Single file
page_countPositive page count when the request contains one document.Single file
document_indexZero-based document order for per-document request mode.Optional
manifestJSON object with batch_id and a documents array matched to file parts by order.Batch metadata
source / scanner_nameHuman-readable capture source displayed in this demo library.Optional
